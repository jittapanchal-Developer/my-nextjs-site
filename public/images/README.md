# /public/images/

Drop your replacement images here. Name them to match the paths in `data/menuItems.js`.

Example replacements:
- burger.jpg       → used for Gourmet Truffle Burger
- iced-latte.jpg   → used for Signature Iced Latte
- croissant.jpg    → used for Dark Chocolate Croissant
- acai-bowl.jpg    → used for Wild Açaí Power Bowl
- avocado-toast.jpg → used for Heirloom Avocado Toast
- matcha-latte.jpg → used for Ceremonial Matcha Latte

Then update the `thumbnail` and `images` arrays in `data/menuItems.js` to use:
  '/images/your-file.jpg'
instead of the Unsplash URLs.
